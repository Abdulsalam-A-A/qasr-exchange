# شركة القصر للصرافة 💱

موقع أسعار الصرف اليومية — بغداد، العراق

## خطوات الرفع على Vercel

### 1. رفع على GitHub
- افتح github.com وسجل دخول
- اضغط **New repository**
- اسمه: `qasr-exchange`
- اضغط **Create repository**
- ارفع جميع ملفات هذا المجلد

### 2. النشر على Vercel
- افتح vercel.com
- سجل دخول بحساب GitHub
- اضغط **Add New Project**
- اختار مشروع `qasr-exchange`
- اضغط **Deploy** ✅
